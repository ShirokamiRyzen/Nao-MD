## Requirements

* NodeJS (v18 up)
* FFMPEG
* Imagemagick
* NodeJS basic knowledges (dont be an idiot)

#### Setup bot on your Machine
```bash
$ git clone https://github.com/ShirokamiRyzen/Nao-MD
$ cd ./Nao-MD
$ npm i
```

---------

## Run

```bash
node index.js

or

node index.js --autocleartmp
```
